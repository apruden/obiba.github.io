  Mica 8.2 Released2014-03-21  

This release fixes bugs for longitudinal studies. You can learn more about the detailshere.

Important note before upgrading Mica

Mica heavily uses Features module to capture configuration data, such as entities, fields, variables, views, permissions, etc into code modules.

In many cases, these Features need to be slightly changed for the specific site you are developing. If you change the base Mica feature, it will be hard to upgrade to a newer version of Mica because your changes would be overwritten.

What you really want to do is just capture your changes to a site-specific "Override" feature and leave the original distribution untouched. To do so, use Features Override module.

Please, refer to documentation before upgrading Mica: Mica Installation Guide
