We are pleased to announce that Mica 9.2 is now available.

This release contains the following features and improvements:

  • Ajax statistics tables in detail page variables
  • SSL version option removed
  • Capability of personalizing Opal connection error messages
  • Permission config made possible to view stats in variable page
  • Deal with studies title if no acronym was not defined
  • Performance enhancement in domain coverage module
  • Added configuration menu to display/hide buttons Study/Datashema coverage
  • Upgraded jQuery DataTable library
  • Added more information in Network search page
  • Added attached studies field in variable page
  • Added capability of searching variables of a study
  • Added capability of searching studies of a network
  • Changed timeline scale for studies with duration greater than 50 years
