We are pleased to announce that Opal 2.2 is now available. Opal is OBiBa’s core database application for biobanks.

The main new features are:

  • rows can be filtered by specifying the "entity filter" of a view in javascript,
  • several Python commands have been added (permissions and data management),
  • new javascript functions have been added,
  • task progress is reported in the web UI,
  • a project can be archived,
  • data indexing can be suspended.
