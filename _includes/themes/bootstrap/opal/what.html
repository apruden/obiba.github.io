Opal is OBiBa’s core database application for biobanks or epidemiological studies. Opal provides all the necessary tools to import, query, analyse and export data, produce and email complex data reports, manage subject’s identifiers, store and access data files and many more. TESTING WITH Vincent

Opal is integrated with Onyx; Studies can seamlessly and securely import collected data using Onyx to Opal. Opal is also strongly integrated with Mica allowing studies to create web data portals offering real-time aggregated reports on the exact number of subjects with specific characteristics or phenotypes.

Opal implements the DataSHIELD method which enables advanced statistical data analysis across multiple Opal instances without disclosing any individual-level data. Learn more >

Opal is built upon a Restful architecture with multiple available clients such as a user-friendly web application, a R and Python command line clients.
