Opal is OBiBa’s core database application for biobanks or epidemiological studies. Opal provides all the necessary tools to import, query, analyse and export data, produce and email complex data reports, manage subject’s identifiers, store and access data files and many more.

Opal is integrated with Onyx and Mica. Studies can seamlessly and securely import data collected with Onyx into Opal. They can also create web data portals with Mica that query Opal databases to obtain real-time aggregated reports on subject's data.

Opal implements DataSHIELD which enables advanced statistical data analysis across multiple studies without sharing and disclosing any individual-level data. Learn more >

Opal is a Restful application with multiple available software clients such as a user-friendly web application, a R and Python command line clients.

Opal is highly scalable and can store very large datasets, thanks to its state-of-the art database technologies built upon MongoDB and ElasticSearch .
