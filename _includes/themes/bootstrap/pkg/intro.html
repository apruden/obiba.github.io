This is the Debian package repository of OBiBa software to automate installation and upgrade. To use this repository, first add the key to your system: 

wget -q -O - https://pkg.obiba.org/obiba.org.key | sudo apt-key add -

Then add the following entry in your /etc/apt/sources.list: 

deb https://pkg.obiba.org stable/

Update your local package index: 

sudo apt-get update
Please note that the package apt-transport-https is required for apt to be able to handle repositories with a https URL. If for any reason you cannot install this package, our debian repository is still accessible through http.
