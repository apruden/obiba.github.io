Our Applications

Collect with Onyx

A comprehensive software application used to manage participant interviews and to collect data in biobank assessment centres. Onyx can be branded with your own colors and logos.

Learn more

Store with Opal

Integrates biobank data from different sources into a secure central data repository. It provides interfaces for managing and querying a data element dictionary and for generating administrative and scientific reports.
